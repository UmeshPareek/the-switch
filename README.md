## Admin — Adding Properties

1. Open the live site
2. Scroll to the bottom of the Properties section
3. Click the tiny **`admin ↗`** link
4. Password: `theswitch2025`
5. Upload property name, location, rent, image
6. Properties save to the visitor's browser (localStorage)

> To make properties persist for all visitors, replace localStorage with a backend (Firebase, Supabase, or a simple JSON file on a server).

## Deploying on GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Source: **Deploy from branch → main → / (root)**
4. Save — site goes live in ~60 seconds

Custom domain: Add a `CNAME` file with `theswitch.in` (or your domain),  
then point your DNS A records to GitHub's IPs.
